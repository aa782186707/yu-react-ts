# react-mobx-ts-antd

A simple empty project build with react、react-router、mobx、antd in typescript.

[Demo](http://evolify.oschina.io/demo/react-mobx-ts/#/)

### usage:

1. `git clone https://github.com/evolify/react-mobx-ts-antd.git <dir-name>`  
2. `cd <dir-name>`
3. `cnpm i` or `npm i`

### run in dev:

`npm run dev`

### build:

`npm run build`
